Vector — OpenFreeMap (MapLibre GL)

Vector tiles, not raster — these maps render via MapLibre GL JS, not Leaflet. Because they're vector, label visibility is a runtime toggle (the symbol layers get hidden/shown live). Raster styles can't do this — labels are baked into the PNGs.
